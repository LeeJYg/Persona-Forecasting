## 종합 결론

1. **DirAcc 0.55 재현 판정:** 계산 방식 차이가 주된 원인
   - exp009 원본(flat 제외): 0.5453 vs 현재 metrics.py(flat 포함): 0.4175 (Δ=+0.1278)
   - 평가 기간(17w→16w) 효과는 Δ=+0.015로 부차적

2. **Track B vs lightgbm 통계 비교:**
   - lightgbm MAE=8.5849, Track B MAE≈8.6~9.7 (seed 분산 큼)
   - Wilcoxon signed-rank test: p=0.4112 → **유의미한 차이 없음**
   - 아이템별 우위: Track B 43/100 vs lightgbm 57/100
   - DirAcc: Track B 0.39 vs lightgbm 0.38 (Track B 소폭 우위)

3. **개선 우선순위:**
   - 스케일 오차(77.1%) >> 패턴 오차(22.9%) → 절대 판매량 예측 개선이 핵심
   - FOODS 카테고리 집중 (고오차 Top-20의 55%)
   - FOODS_3_120 등 극단 outlier (실제 246 → 예측 9) 처리 방안 필요
