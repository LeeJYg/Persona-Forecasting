## 파트 2: Diffuse Attention 아이템 분석

### 배경
- Sparse attention 아이템: MAE 3.37
- Diffuse attention 아이템: MAE 13.65
- Spearman r(entropy vs MAE) = 0.607

### 실험 2A: Diffuse 아이템 프로파일링

cold 100개를 attention entropy 기준으로 상위 25개(diffuse)와 하위 25개(sparse)로 분류하라.

각 그룹에 대해:
1. 카테고리 분포 (FOODS/HOBBIES/HOUSEHOLD 비율)
2. 평균 주간 판매량
3. 가격 분포
4. attention weight의 top-1 페르소나 집중도 (top-1 weight)
5. 실제 판매량의 주간 변동성 (std / mean = CV)
6. embedding norm (residual 벡터의 L2 norm 평균)

### 실험 2B: Diffuse → Sparse 유도 가능성

attention을 더 sparse하게 만드는 방법을 테스트:

**2B-1: Temperature Scaling**
attention softmax에 temperature를 적용: softmax(scores / T)
T = [0.1, 0.3, 0.5, 1.0(현재), 2.0]
각 T에서 cold MAE, DirAcc 측정.

**2B-2: Top-K Attention**
softmax 후 상위 K개 페르소나만 유지, 나머지는 0으로 마스킹.
K_attn = [3, 5, 10, 20, 50(현재)]
각 K에서 cold MAE, DirAcc 측정.

주의: 이 두 방법은 **inference 시에만** 적용한다 (학습은 기존과 동일). 학습 시에도 적용하는 변형은 파트 3에서.

## 파트 3: Loss Function 및 학습 최적화

### 3A: 방향성 Loss 추가

현재: L1 loss (MAE)만 사용.
DirAcc을 직접 최적화하기 위해 방향성 penalty를 추가한다.

```python
# 주간 변화 방향 일치 loss
def direction_loss(pred, target):
    # pred, target: (N, n_weeks)
    pred_diff = pred[:, 1:] - pred[:, :-1]      # (N, n_weeks-1)
    target_diff = target[:, 1:] - target[:, :-1]
    # 방향 불일치 시 penalty
    sign_match = torch.sign(pred_diff) * torch.sign(target_diff)  # 일치하면 +1, 불일치하면 -1
    return -sign_match.mean()  # 최소화 → 방향 일치 최대화

total_loss = mae_loss + lambda_dir * direction_loss
```

lambda_dir = [0.0(현재), 0.1, 0.5, 1.0, 2.0]
각각에서 cold MAE + DirAcc 측정.

주의: direction_loss의 gradient가 sign 함수 때문에 0이 될 수 있다. 이 경우 soft sign (tanh)으로 대체:
```python
pred_diff_soft = torch.tanh(pred_diff * 10)  # scale factor 조절
target_diff_soft = torch.tanh(target_diff * 10)
sign_match = pred_diff_soft * target_diff_soft
```

### 3B: Hyperparameter Grid Search

현재 설정: lr=1e-3, weight_decay=1e-4, bottleneck=64, dropout=0.1, epochs=500.

다음 조합을 grid search:
- lr: [5e-4, 1e-3, 2e-3]
- bottleneck: [32, 64, 128]
- epochs: [300, 500, 1000]

총 27개 조합 × 3 seeds = 81회 학습.
평가: cold MAE + DirAcc. Pareto front (MAE ↓, DirAcc ↑) 보고.

### 3C: 원본 + Residual 결합

원본 embedding과 residual embedding을 concatenate하여 사용:
input = concat(raw, residual) → (N, 50, 10240)
또는 두 개의 별도 attention을 사용하고 결합:
- attn_raw → context_raw (5120)
- attn_residual → context_residual (5120)  
- head(concat(context_raw, context_residual)) → (n_weeks)

이 방식이 원본의 DirAcc(0.55)와 residual의 MAE(8.71)를 동시에 얻을 수 있는지 테스트.
