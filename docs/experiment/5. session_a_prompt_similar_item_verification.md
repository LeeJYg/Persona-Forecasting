# 세션 A 프롬프트: 유사 아이템 기반 파이프라인 핵심 가정 검증 (exp015)

## 배경

exp014 결과 요약:
- 프롬프트 변형(V4-A/V4-B): 효과 없음 → 프롬프트 재추출 경로 폐기
- Residual embedding: 카테고리 분화 안정(diff rate 1.00), N=250에서 MAE 55.8 (원본 62.5 대비 -10.7%)
- Residual + 데이터 확장의 조합이 유망한 신호

새로운 아이디어: 모든 cold item에 하나의 global head를 적용하는 대신,
**각 cold item과 유사한 warm items를 선별하여 item-specific 예측**을 한다.
유사성은 residual embedding 공간에서 정의한다.

이 아이디어를 구현하기 전에 핵심 가정을 검증한다.

---

## 검증 1: Residual Embedding 유사성 ↔ 판매량 유사성 상관

### 목적
Residual embedding 공간에서 가까운 아이템들이 실제로 비슷한 판매량을 가지는가?
이것이 성립하지 않으면 "유사 아이템 선별" 자체가 무의미하다.

### 방법

warm 300개의 residual embedding을 사용한다.
각 아이템의 residual embedding = 50개 페르소나 residual의 평균 (즉, item-mean subtracted 후 persona 평균).
→ 잠깐, 이렇게 하면 0벡터가 된다. **대신 다음을 사용:**

**아이템 표현 방법 3가지를 모두 테스트:**
(a) 원본 mean-pooled: warm_raw.mean(dim=1) → (300, 5120)
(b) 원본 per-persona를 flat: warm_raw.reshape(300, 50*5120) → (300, 256000) — 차원이 너무 크면 PCA(64)로 축소
(c) Residual의 per-persona를 flat + PCA(64): (warm_raw - item_mean).reshape(300, 50*5120) → PCA(64) → (300, 64)

각 표현에 대해:
1. 300×300 pairwise distance matrix를 계산 (cosine distance)
2. 300×300 판매량 차이 matrix를 계산: |mean_weekly_sales_i - mean_weekly_sales_j|
3. 두 matrix 간 Mantel test (행렬 상관) 또는 단순히 각 아이템의 K-nearest neighbors (K=5,10,20)의 **판매량 std**를 계산

### 핵심 판단 기준
- K-NN의 판매량 std가 전체 std보다 **현저히 낮으면** → embedding 유사성이 판매량을 반영 → 유사 아이템 접근 정당화
- 차이가 없으면 → embedding이 판매량 정보를 담지 않음 → 유사 아이템 선별에 다른 기준 필요 (카테고리+가격 등)

## 검증 2: K-NN 기반 예측 (warm 내부 leave-one-out)

### 목적
유사 아이템 기반 예측이 global head보다 나은지 warm 내부에서 먼저 확인한다.

### 방법

warm 300개에 대해 leave-one-out:
- 각 아이템 i를 제외한 299개에서 K-nearest neighbors를 찾는다
- K개 neighbors의 판매량 **가중 평균** (거리 역수 가중)으로 i의 판매량을 예측한다
- K = [5, 10, 20, 50]

비교 대상:
- Global mean (300개 전체 평균 = 96.23): MAE 상한
- Global Attention+Bottleneck (exp013 A, N=250): MAE 62.5 (원본) / 55.8 (residual)

유사성 기준 3가지를 비교:
(a) Residual per-persona flat + PCA(64)에서의 cosine distance
(b) 원본 mean-pooled에서의 cosine distance
(c) 카테고리 + 가격 기반 (embedding 무관): 같은 카테고리 내에서 가격이 가장 가까운 K개

### 핵심 판단 기준
- (a) 또는 (b)의 MAE가 (c)보다 낮으면 → embedding 유사성이 카테고리+가격보다 나은 유사성 기준
- (a)/(b) 모두 (c)보다 나쁘면 → embedding 유사성은 쓸모없고 카테고리+가격이면 충분
- 어떤 K-NN이든 global Attn+Bottleneck(55.8)보다 나으면 → 유사 아이템 접근이 유망
- K-NN이 Global mean(96.23)과 비슷하면 → 유사 아이템 접근 자체가 무의미

## 검증 3: 유사 아이템 기반 Attention Head (핵심 실험)

### 목적
가장 중요한 실험. 검증 2가 단순 K-NN 평균이라면, 이것은 **K개 유사 아이템으로 attention head를 학습하고 대상 아이템을 예측**하는 것이다.

### 방법

warm 300개에서 leave-one-out:
- 아이템 i에 대해, residual embedding 기준 K-nearest warm items를 찾는다
- 이 K개의 (residual_embedding, weekly_sales) 쌍으로 Attention+Bottleneck head를 학습
- 학습된 head로 아이템 i의 residual embedding → 판매량 예측

K = [10, 20, 50]
(K < 10이면 학습 불가능할 수 있으므로 최소 10)

### 핵심 판단 기준
- Leave-one-out MAE < global Attn+Bottleneck MAE(55.8) → 유사 아이템 + attention 조합이 유망
- Leave-one-out MAE ≈ global → 유사 아이템 선별의 이점이 없음

### 현실성 검토
- K=10~50으로 attention head를 학습하면 파라미터 대비 샘플이 극히 부족할 수 있다.
- 이 경우 대안: global head를 300개 전체로 학습하되, 예측 시 K-nearest의 **판매량 통계(mean, std)**로 output을 rescale한다.
- 즉: pred_i = global_head(residual_i) × (mean_sales_of_K_neighbors / mean_sales_of_all_warm)
- 이 rescale 방식도 함께 테스트하라.
